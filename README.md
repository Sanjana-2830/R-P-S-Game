# workflow of project
1 input from user (rock , paper ,scissor)
2 computer choice 
3 result


# cases
* A Rock 
Rock - ROck = tie
rock - scissor = rock win
rock - paper = paper win

* B Paper 
paper - paper = tie
paper - scissor = scissor win
paper - rock = paper win


* C Scissor
Scissor - scissor = tie
scissor - paper = scissor win
scissor - rock = rock win
